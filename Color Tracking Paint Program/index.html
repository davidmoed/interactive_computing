Welcome to the interactive Coloring Program

Use the up and running video, or switch the menu from 'video' to 'still images' and press C to take a picture from your camera.
Click on anything in the video/image to save its color and start painting with it!

To change the shape to paint with, use the dropdown menu, or press 'A' for circles, 'S' for squares, and 'D' for triangles.

To save your sketch press 'X'

For this project I wanted to combine the color capture with a simple painting program, that can be expanded to allow the user a wide range of shapes, all with a color of their choosing. This program allows the user to hold up any colored item to their computer camera and capture it. The user can currently draw either circles, squares, or triangles in this color.

The user can choose to capture either with constantly capturing video with their camera, or capture stills. The stills are useful because the user can paint over them, and they require less computing power, allowing the program to run faster.
